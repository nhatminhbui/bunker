.31 MB misspell.txt This file contains 30,042 misspellings of 4,897 words, originally derived from Mitton Roger's birkbeck file. I excluded some misspellings which themselves are correct words. It includes the results of spelling tests and errors from free writing, taken mostly from schoolchildren, university students or adult literacy students. Each line starts with a correct word, followed by its misspellings on the same line, using space as a delimiter.
